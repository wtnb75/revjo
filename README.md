# Reverse-jo

- revjo convert $(jo hello=world)
  - `hello=world`
- jo hello=world | revjo convert | jo
  - `{"hello":"world"}`

# Install

- `pip install revjo`
- or... `whalebrew install ghcr.io/wtnb75/revjo`
